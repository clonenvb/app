📦 Công cụ Split & Zip (v1.4) – Python + PyQt5

🎯 Tính năng chính:
--------------------
✅ Tách ảnh thành 9 phần (3x3) và xuất ra thư mục + file ZIP
✅ Tự động nén và mở file ZIP sau khi hoàn tất
✅ Xem nhật ký quá trình xử lý ảnh
✅ Nút "Chạy lại" giúp làm lại từ đầu
✅ Cho phép mở ảnh vừa xử lý chỉ bằng 1 click
✅ Hiển thị phiên bản hiện tại và tự kiểm tra cập nhật từ GitHub

🧹 Dọn rác thông minh:
-----------------------
▶️ Dọn nhanh:
  - Chọn số ngày giữ lại (1–7 ngày)
  - Tự động xóa các tệp cũ hơn số ngày đã chọn
  - Hiển thị danh sách file đã dọn

▶️ Dọn nâng cao:
  - Hiển thị danh sách các file đủ điều kiện xóa
  - Cho phép tick chọn từng file để xoá thủ công

📖 Hướng dẫn cập nhật từ xa:
-----------------------------
- Nội dung hướng dẫn có thể được cập nhật online từ file README.txt trên GitHub
- Người dùng nhấn "Hướng dẫn" để xem hướng dẫn mới nhất mà không cần cập nhật toàn bộ ứng dụng

🛠 Yêu cầu cài đặt:
--------------------
- Python 3.x
- PyQt5
- Pillow (PIL)

👉 Cài thư viện:
pip install PyQt5 Pillow
